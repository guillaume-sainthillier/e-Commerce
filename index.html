e-Commerce

e-Commerce est un site de simulation de vente de materiel informatique dans le cadre d'un projet de l'I.U.T Informatique de Blagnac

Déploiement

Pour déployer l'application, veuillez suivre ces indications:

$ cd path/to/www
$ git clone https://github.com/guillaume-sainthillier/e-Commerce.git

Installation

Adaptez les variables host,user,pass et db du fichier includes/utils.php à la configuration de votre serveur.
Importez les fichier sql/e-Commerce.sql et sql/trigger.sql dans votre base de données.

Pré requis

Vous devez avoir installé:

  • un serveur apache et php
  • une base de données : mysql ou postgresql

En fait, reportez vous aux installations de :

  • sous windows : wamp (http://www.wampserver.com/) ou easyphp(http://easyphp.fr/)
  • sous linux : lamp

Si vous utilisez postgre sql, voir install sur le site : http://www.postgresqlfr.org/

Si vous debutez, il est plus simple de garder mysql qui est packagee avec easyphp ou wamp.
